Drop your image files into the matching folder here, then set the path
in `lib/site-images.js` (e.g. `/images/team/photo-1.jpg`).

- logos/         → client logo images (ClientLogos.jsx)
- team/          → Who We Are photo collage (WhoWeAre.jsx)
- approach/      → B2B card portrait (ApproachCards.jsx)
- our-works/     → portfolio grid tiles (OurWorks.jsx)
- testimonials/  → client avatar photos (Testimonials.jsx)
- video/         → poster image / video file for the hero video (TestimonialVideo.jsx, VideoModal.jsx)

See EDITING-GUIDE.md in the project root for full instructions.
